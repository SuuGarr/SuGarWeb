Senior Project

Detection of Solidification crack and Prediction formation in GTAW of
Aluminum alloy using Artificial Intelligence base Method on an infrared
thermal imaging.

A research project titled ‘Detection of Solidification crack and Prediction
formation in GTAW of Aluminum alloy using Artificial Intelligence base
Method on an infrared thermal imaging.' The project applies knowledge in
Machine Learning, Machine Vision, and welding technology of materials. It
aims to detect and predict cracks in aluminum welds using artificial
intelligence and thermal imaging cameras to enhance the quality and safety
of materials used in various industries.

Awards: Best Oral Presentation: ISPC Conference 2024 organized by KMUTT (Thailand)
• Computer language: Python, JavaScript, CSS, HTML

Intelligent Prosthetic Arm.

Intelligent Prosthetic Arm works by using ‘Flex sensor’ to receive bending
signals from Arm impaired people’s muscles around the Cubital Fossa or
elbow pit, then use the bending value to control the Intelligent
Prosthetic Arm. The Autodesk Inventor program has been used for
planning and designing the prosthetic arm. After the developer
had Mr.Chawakorn Thummameta (Arm impaired people) test the Intelligent
Prosthetic Arm for assisting in daily life, the result was satisfying,
the user could easily control the prosthetic arm and this innovation
could help with the user’s daily tasks.

• Computer language: C
