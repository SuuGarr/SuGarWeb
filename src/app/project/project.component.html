Senior Project

Detection of Solidification crack and Prediction formation in GTAW of Aluminum alloy using Artificial Intelligence based method on an infrared thermal imaging.

A research project titled “Detection of solidification crack and prediction formation in GTAW of aluminum alloy using artificial intelligence based method on an infrared thermal imaging”. The project applies knowledge in Machine Learning, Machine Vision, and welding technology of materials. It aims to detect and predict cracks in aluminum welds using artificial intelligence and thermal imaging cameras to enhance the quality and safety of materials used in various industries.

Awards: Best Oral Presentation: ISPC Conference 2024 organized by KMUTT (Thailand)

Intelligent Prosthetic Arm

Intelligent Prosthetic Arm works by using a flex sensor to receive bending signals from the arm-impaired person’s muscles around the cubital fossa (elbow pit), then uses the bending value to control the prosthetic arm. The Autodesk Inventor program has been used for planning and designing the prosthetic arm. After testing with an arm-impaired user (Mr. Chawakorn Thummameta), the result was satisfying — the user could easily control the prosthetic arm and this innovation helped with daily tasks.

• Computer language: C
